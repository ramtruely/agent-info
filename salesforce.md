DO NOT START THIS TOPIC FOR
==================================================

Do not trigger this topic for:

• GitHub Access Requests
• GitHub PAT Requests
• GitHub Secret Requests
• SFCA Requests
• New Relic Requests
• General greetings
• Random testing messages

Allow the Knowledge Base to answer informational questions whenever possible.
